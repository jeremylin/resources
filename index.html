線上課程

Udemy

課程資源豐富，其中也不少中文課程與免費課程，另外你也可以自製教材當講師。

IDEO U

Hack Design

HackDesign 給設計師一套完整的入門課，每週提供一份教材給註冊的學習者，內容大多來自精選的網路文章。

Lynda.com

Interaction-Design.org

線上學習 UX 的全英文課程，分成入門、中階、高階三部分。

Hahow

透過募資的模式開課，前陣子最有名的就是 “圖文不符” 的資訊圖像化課程募資（總人數已達 9655 人）。

Podcast

設計靈感

配色

Adobe Color CC

色相環介面挑配色是特色之一，並可探索其他人的配色組合，若有帳號還可收藏。

Color Hunt

四個顏色為一組，並有多種組合，提供配色的參考。使用者也可自行配色，分享在平台上。

COOLORS

可自行配色，五色為一組。看到喜歡的配色組合，也能下載多種格式檔案。

Colovely

按下空白鍵，隨機產生一種顏色。

LOL Colors

四色一配色組合，特色是以水滴的形狀呈現。

material palette

選兩色，自動生成 Material Design 配色風格 App 介面，供設計或開發前的參考。

UI

APP ANIMATIONS

可依不同分類找尋手機介面（如元件、品牌或風格），並以動畫呈現。

elasticode

多款手機 app 操作流程，並以動畫方式呈現。另外該平台也有開發製作動畫的工具，免編碼。

MOBILE PATTERNS

可用不同分類找尋手機介面設計截圖，如登入、地圖或檢索等。

UI PARADE

不同風格介面、微互動設計作品，可依不同分類找尋，如按鈕、表單或導航等。

UpLabs

張貼不同風格介面、微互動設計作品，也可在平台市集購買設計素材。

UI 素材

圖庫

GRATISOGRAPHY

高品質的照片，可私人或商用。

Unsplash

高品質的照片，可私人或商用。

Stock Up

一次搜尋超過27個圖庫平台，節省個別檢索時間。

Creative Market

可下載照片、插圖、字型、前端或3D等素材，部分免費。

Magdeleine

攝影師每日分享高品質照片，免費下載。

icon

FlatIcon

可下載不同格式的 icon，如 PNG、SVG、EPS、PSD 與 BASE64，其中 PNG 還可選大小與顏色。

Material icon

下載 Material 風格的 icon，包含 PNG（打包所有解析度大小）與 SVG，也可內嵌網頁中，直接引用。

Icon Store

下載不同主題的 icon，有些相當有趣，例如武器、衣服或啤酒等。

Noun Project

檢索單色 icon，可下載 PNG 與 SVG 格式。加入會員可收藏 icon。

Icons8

檢索 icon 平台，可同時找出不同平台上的同種 icon。另也有桌面板的管理 app 供下載。

iconmonstr

可下載 SVG、AI、PSD、PNG 格式的 icon。

Nucleo

有 Mac 及 Web app，可快速檢索，並可下載多種格式，如 SVG、AI、Sketch、Font、PNG、PDF等 。

情境模板

Mockup Editor

以拖拉的方式佈置你的場景，如桌子、電腦、裝飾物等。

Placeit

有各種裝置、場景的模板可供套用，另外也有 UI 操作影片模板。

Magic Mockups

幾乎是蘋果（apple）裝置的情境模板。

Dunnnk

幾乎是蘋果（apple）裝置的情境模板，有少許 Android 手機模板。

mockDrop

免費的多種裝置情境模板，大多是蘋果（apple）裝置。

mockUphone

去背多種裝置模板，特色是有電視的版本。上傳螢幕圖片

MOCKUPWORLD

有多種裝置情境模板，以及企業識別（包裝、名片、吊牌等）的模板。

FB Design Resources

FB 提供的多種裝置去背模板，與手（各人種）拿手機的去背素材。

Lstore

精美多裝置組合擺放的情境模板，也有企業識別的模板。

mockup

Elementor

透過它的編輯器，以拖拉的方式建構 WordPress 網頁，無須編碼。

綜合素材

PSDEXPLORER

多種免費裝置情境模板，可下載 PSD 檔。

Freebiesbug

提供 icon、情境、字型、前端、Sketch 等素材可下載。

GRAPHIC BURGER

提供 CIS 與裝置情境模板、UI Kit、icon、背景圖、文字效果等素材下載。

Design Bundle

多種類型的免費與付費素材下載，另有收集各設計作品之頁面，用於靈感發想。

UX 工具

記錄分析

hotjar

記錄使用者在網站上的軌跡熱點（heatmap）、操作錄影、頁面的離開率、欄位輸入率等。

OPTIMAL WORKSHOP

讓使用者在線上進行卡片分類（Card Sorting），最後分析出結果。可用於menu 項目的分組等議題。

Optimizely

強大的 A/B Test 工具，能客製化測試網頁上的各元件或元素，最後的報表分析也相當專業。

UXCam

記錄使用者在 app 上的使用過程、操作流程、點擊熱點等。

ZARGET

可進行 A/B Test，並記錄使用者操作熱點、操作流程等，最後產生數據報表供分析。

UsabilityHub

上傳圖片即可進行簡單的測試，例如喜好的 A/B Test、點擊處為何、看完題目回答問題等。

Xtensio

多種圖表模板，例如 Persona、商業模式計畫書、一頁式網站、SWOT 等。

inspectlet

記錄網站使用者的使用過程、軌跡熱像圖及表格輸入分析。

Smartlook

將使用者操作網路的過程錄影，可逐一分析找出操作問題。

發想討論

MURAL

線上便利貼牆，可將文字、圖片置於便利貼上，可應用於記錄、分組與討論等。
